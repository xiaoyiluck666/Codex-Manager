# Минимальные инструкции по доступу к центру плагинов

Это минимальное руководство для быстрого подключения стороннего или внутреннего источника плагинов. Если источник возвращает данные в формате ниже, его можно подключить к центру плагинов CodexManager.

## 1. Что вы хотите получить?

- Внешняя маршрутизация: `/plugins/`
- Режим рынка: `builtin` / `private` / `custom`
- Формат данных: список плагинов JSON.
- Метод выполнения: Rhai Скрипт.

## 2. Минимальная рыночная доходность

Возвращает массив верхнего уровня или возвращает:

```json
{ "предметы": [] }
```

Каждый плагин должен иметь как минимум:

```json
{
  "id": "очистка заблокированных аккаунтов",
  "name": "Очистить заблокированные аккаунты",
  "версия": "1.0.0",
  "scriptBody": "fn run(context) { ... }",
  "permissions": ["accounts:cleanup"],
  "задачи": [
    {
      "идентификатор": "запустить",
      "name": "Запланированная автоматическая очистка",
      "точка входа": "запустить",
      "scheduleKind": "интервал",
      «интервалСекунды»: 60,
      «включено»: правда
    }
  ]
}
```

## 3. Минимальное поле

| Поле | Требуется | Описание |
| --- | --- | --- |
| `id` | Да | Уникальный идентификатор |
| `name` | Нет | Имя, по умолчанию возвращается к `id` |
| `version` | Нет | Номер версии, по умолчанию `0.0.0` |
| `scriptBody` | Выберите один | Встроить скрипт напрямую |
| `scriptUrl` | Выберите один | Адрес скрипта |
| `permissions` | Нет | Список разрешений |
| `tasks` | Нет | Определение задачи |

Рекомендуется добавить:

- `manifestVersion`
- `category`
- `runtimeKind`
- `tags`

## 5. Rhai Минимальная доступная функция

Если плагин претендует на эти разрешения, он может использовать соответствующие функции:

- `settings:read` -> `get_setting(key)`, `list_settings()`
- `network` -> `http_get(url)`, `http_post(url, body)`
- `accounts:cleanup` -> `cleanup_banned_accounts()`, `cleanup_unavailable_free_accounts()`

Публичные функции:

- `log(message)`

##6. Входной вывод

Если вы:

- Официальный выбранный рынок напрямую возвращает встроенный JSON с той же структурой.
- Частный рынок предприятия, возвращает частный склад в формате JSON.
- Чтобы настроить склад, просто сохраните `scriptBody` или `scriptUrl`.

В настоящее время наиболее стабильным методом подключения является «сначала список, потом сценарий». Не помещайте тяжелую логику непосредственно в Rhai.
