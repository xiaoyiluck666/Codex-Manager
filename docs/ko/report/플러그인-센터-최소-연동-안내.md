# 플러그인 센터에 대한 최소 접근권한 안내

이 문서는 타사 또는 내부 플러그인 소스를 빠르게 연결하기 위한 최소 연동 가이드입니다. 아래 필드와 인터페이스 형식에 맞춰 데이터를 반환하면 CodexManager 플러그인 센터와 연동할 수 있습니다.

## 1. 무엇을 선택하고 싶으신가요?

- 프런트엔드 라우팅: `/plugins/`
- 시장 모드: `builtin` / `private` / `custom`
- 데이터 형식: 플러그인 목록 JSON
- 실행방법 : Rhai 스크립트

## 2. 최소 시장 수익률

최상위 배열을 반환하거나 다음을 반환합니다.

``json
{ "항목": [] }
````

각 플러그인에는 최소한 다음이 있어야 합니다.

``json
{
  "id": "정리 금지 계정",
  "name": "금지된 계정 삭제",
  "버전": "1.0.0",
  "scriptBody": "fn 실행(컨텍스트) { ... }",
  "권한": ["계정:정리"],
  "작업": [
    {
      "id": "실행",
      "name": "예약된 자동 청소",
      "entrypoint": "실행",
      "scheduleKind": "간격",
      "간격초": 60,
      "활성화됨": 사실
    }
  ]
}
````

## 3. 최소 필드

| 필드 | 필수 | 설명 |
| --- | --- | --- |
| `id` | 예 | 고유 ID |
| `name` | 아니요 | 이름, 기본값은 `id` |
| `version` | 아니요 | 버전 번호, 기본값 `0.0.0` |
| `scriptBody` | 하나를 선택하세요 | 스크립트 직접 삽입 |
| `scriptUrl` | 하나를 선택하세요 | 스크립트 주소 |
| `permissions` | 아니요 | 권한 목록 |
| `tasks` | 아니요 | 작업 정의 |

다음을 추가하는 것이 좋습니다.

- `manifestVersion`
- `category`
- `runtimeKind`
- `tags`

## 5. Rhai 최소 사용 가능 기능

플러그인이 이러한 권한을 적용하는 경우 해당 기능을 사용할 수 있습니다.

- `settings:read` -> `get_setting(key)`, `list_settings()`
- `network` -> `http_get(url)`, `http_post(url, body)`
- `accounts:cleanup` -> `cleanup_banned_accounts()`, `cleanup_unavailable_free_accounts()`

공개 기능:

- `log(message)`

## 6. 입학결론

귀하가 다음과 같은 경우:

- 공식 선정 시장, 동일한 구조로 내장된 JSON을 직접 반환
- 기업용 프라이빗 마켓: 프라이빗 소스 JSON 반환
- 커스텀 소스를 사용하려면 `scriptBody` 또는 `scriptUrl`을 유지하세요.

현재 가장 안정적인 연결 방법은 "목록 먼저, 스크립트 나중에"입니다. Rhai에 무거운 논리를 직접 쌓지 마십시오.
